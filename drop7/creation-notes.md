# Drop7 Conversion Notes

## Original Project

Drop7-style puzzle rules; no external source repository is bundled.

## Game Summary

Number puzzle played on a 7x7 board. The player drops numbered or solid discs into columns, clears numbered discs when their value matches a contiguous row or column count, cracks solid discs through adjacent clears, and scores chain reactions.

## Conversion Scope

The implementation is self-contained and uses no personal source material or private project paths.
