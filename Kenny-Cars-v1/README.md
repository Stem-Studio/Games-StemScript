# Kenny Cars v1

## Original Project

StemStudio scene export "Kenny Cars v1" (private scene identifiers removed).

## What This Game Does

Arcade truck-racing scene built from modular track pieces. The player drives a truck around a closed course with terrain decorations, collision walls, engine/skid/impact audio, a chase camera, and optional NPC traffic.

## Notes

This project keeps only portable scene/gameplay details; private scene IDs and owner IDs have been removed.
