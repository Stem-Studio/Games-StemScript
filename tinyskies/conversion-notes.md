# TinySkies Conversion Notes

## Original Project

TinySkies web app, deployed at https://tinyskies.vercel.app (local source path removed).

## Game Summary

Multiplayer globe-flight game. Players fly stylized aircraft around a small procedural planet, steer with smoothed flight physics, use a chase camera, and fire paintballs. This port focuses on flight, globe visuals, and paintball behavior.
