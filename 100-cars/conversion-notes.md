# 100 Cars Conversion Notes

## Original Project

StemStudio scene export "100 cars" (private scene identifiers removed).

## Game Summary

Arcade driving game using the same modular racing-kit assets as Kenny Cars. The player controls a truck on a track while NPC trucks loop around the course with vehicle audio, collisions, and camera follow behavior.

## Conversion Scope

This project keeps only portable scene/gameplay details; private scene IDs and owner IDs have been removed.
