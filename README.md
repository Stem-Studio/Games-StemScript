Stem Studio Games Repository
-
We've recently open-sourced Stamp Studio. This repository contains a selection of the games we've built with Studio. These examples are indicative—we'll be adding more over time.
Please note that we're in the process of transitioning our network layer into a stubbed interface that other developers can build on, so some of these games may not work as expected. The purpose of these examples is to give you a sense of how you can build your own games and extend Studio.
